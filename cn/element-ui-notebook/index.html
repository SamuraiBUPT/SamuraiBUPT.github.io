el-row & el-col 与页面布局

Q:我使用el-card标签创建了这个卡片，然后设置了el-row，在这一行里面设置了两个列，也就是el-col。这一切整体的效果是：让这个卡片分为两半，左边部分放置图片，而右边部分放我的登陆表单。现在的情况是：我的两个col呈现上下关系，并不是左右关系。两个el-col无论怎样调整，都是一个在上面一个在下面。

当你使用 element-ui 的 el-row 和 el-col 组件时，你需要为 el-col 组件设置 span 属性来指定它应该占据的列数。默认情况下，el-row 的栅格系统基于 24 列。如果你没有为 el-col 设置 span，则它可能会默认占据全部的 24 列，这就是为什么你的两个 el-col 元素会堆叠在一起的原因。

为了使两个 el-col 元素并排显示，你可以为它们分别设置 span 属性。例如，如果你希望它们各自占据一半的空间，你可以设置 span="12"：

这个el-row, el-col就是element-ui的布局工具。通过基础的 24 分栏，迅速简便地创建布局。

使用列创建基础网格布局

通过 row 和 col 组件，并通过 col 组件的 span 属性我们就可以自由地组合布局。

支持列间距

行提供 gutter 属性来指定列之间的间距，其默认值为0。

列偏移

通过制定 col 组件的 offset 属性可以指定分栏偏移的栏数。

对齐方式

通过justify 属性来定义子元素的排版方式，其取值为start、center、end、space-between、space-around或space-evenly。

不过注意，是子元素。

display: flex

display: flex 是 CSS 的一个属性值，用于将一个元素的显示模式设置为 Flexbox，也就是灵活的盒子模型。Flexbox 是一个用于在复杂布局和对齐内容中设计更有效布局、对齐内容和分配空间的模型，即使当项目的大小未知或动态变化时也是如此。

默认方向：在不设置 flex-direction 的情况下，默认的子元素布局方向为横向（row）。
