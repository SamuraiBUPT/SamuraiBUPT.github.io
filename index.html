About C++

C++ is not a perfect language, which was also ridiculed by many famous ones including Linus Torvalds. The readability of a language may affect the cooperation of programmers to a certain degree. Compared with other languages like python and golang, cplusplus may not be the best choice to cope with each other.

The only goal of C++ is pursuing extreme performance. Every operation matters to the efficiency of the whole system. It is more of a work of art than a code project. The programmer should manage the memory carefully to avoid crash, which is a challenging task, but worth spending time for.

Coping with ptrs and managing memory is a really romantic job. I will keep going in the path of C++ develpoing.
